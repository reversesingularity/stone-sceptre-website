## Universal Fill-In-The-Blank System

**Status:** PRODUCTION TEMPLATE — Generic, reusable for every chapter
**Aesthetic:** Balanced Splinter Cell tactical atmosphere + Ancient Future bioluminescence
**Format:** Chapter start pages — tall vertical composition

---

## HOW TO USE THIS TEMPLATE

1. Copy the **BASE PROMPT** below
2. Replace every `[BRACKETED FIELD]` with chapter-specific content
3. Use the **Field Guide** table to understand what each field expects
4. Keep all **Fixed Aesthetic Anchors** unchanged — they maintain visual consistency across the entire book
5. Paste the completed prompt into Midjourney as `/imagine [your completed prompt]`

---

## BASE PROMPT

```
/imagine [SHOT_TYPE], [CHAPTER_SUBJECT], [KEY_VISUAL_ELEMENT], [ENVIRONMENT], [MOOD_MODIFIER], deep space blacks and amber bioluminescent light, [GEOMETRY_ELEMENT], balanced Splinter Cell tactical atmosphere and Ancient Future aesthetic, cinematic vertical composition, hyper-detailed concept art --ar 9:16 --v 6.1 --s 250 --no text, words, letters, [EXCLUSIONS]
```

---

## FIELD GUIDE

| Field | What to Put Here | Good Examples | Avoid |
|---|---|---|---|
| `[SHOT_TYPE]` | Camera angle and framing | `dramatic low-angle shot` / `silhouette shot` / `aerial shot` / `close-up detail shot` / `wide establishing shot` | Vague terms like "nice shot" |
| `[CHAPTER_SUBJECT]` | The central visual subject of the image | `lone warrior` / `ancient stone tablet` / `woman in darkness` / `flooded world` / `burning city` | Multiple competing subjects |
| `[KEY_VISUAL_ELEMENT]` | The single defining visual detail that makes this chapter unique | `bronze sword with barely-visible Enochian script` / `twenty amber lines forming constellation` / `fountains of the deep erupting` | More than 2 key elements |
| `[ENVIRONMENT]` | The physical location or setting | `Antarctic research station interior` / `deep ocean trench` / `ruined Mesopotamian city` / `absolute void` / `Martian surface at dusk` | Abstract non-visual descriptions |
| `[MOOD_MODIFIER]` | The atmospheric tone and lighting condition | `near-total darkness with single amber light source` / `volcanic amber light from below` / `bioluminescent deep water` / `volumetric fog and rain` | Bright, cheerful, or daytime descriptions |
| `[GEOMETRY_ELEMENT]` | The sacred geometry detail that ties to the Ancient Future aesthetic | `fracture patterns in stone` / `constellation of names` / `tectonic rift lines` / `sacred proportions in architecture` / `orbital paths as light trails` | Literal descriptions of geometry |
| `[EXCLUSIONS]` | What MJ must NOT generate | `no fantasy creatures` / `no bright colors` / `no modern elements` / `no human figures` | Leaving this blank |

## ITERATION WORKFLOW

When MJ generates a result you want to refine:

1. **Vary (Subtle)** — Small changes to the same composition
2. **Vary (Strong)** — Significant reinterpretation of the same prompt
3. **Remix Mode** — Edit the prompt while keeping the composition
4. **`/describe`** — Drop an image you like into MJ to reverse-engineer its prompt style

**Common fixes:**
| Problem | Solution |
|---|---|
| Too bright / colorful | Add `no bright colors, no warm tones` to `--no` block |
| Geometry too literal | Replace with more evocative descriptor from Geometry Reference Palette |
| Subject too small | Add `subject filling frame` or change shot type to `close-up` |
| Too fantasy / medieval | Add `no fantasy elements, no medieval aesthetic, no swords and sorcery` to `--no` block |
| MJ adds text/glyphs | Ensure `--no text, words, letters` is in every prompt |
| Too sci-fi / neon | Add `no neon, no sci-fi holographics, no laser effects` to `--no` block |
