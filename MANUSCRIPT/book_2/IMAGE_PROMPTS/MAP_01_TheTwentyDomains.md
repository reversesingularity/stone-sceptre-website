# Map 1 — The Twenty Domains
## DALL-E / Microsoft Designer Image Prompt

**Map Description:** A global projection showing all twenty Watcher domain locations scattered across six continents, with the transit network (planar gate connections) overlaid. Each domain marked with its Watcher chief's designation.

---

### Prompt

A stylised antique-modern hybrid world map on aged parchment with precise cartographic line-work. The map uses a Mercator-style projection showing all continents. Twenty points are marked across six continents with glowing golden circular sigils — each sigil containing a unique geometric pattern representing a Watcher chief's acoustic signature. Faint golden lines connect certain points, showing the planar gate transit network. Key locations are emphasised with larger sigils: Vatican City (Shemyaza — the Captain), Carpathian Mountains (Armaros — Ward-Craft), Greenland (planar gate hub), Antarctica (Gadreel/Dudael prison), and a line extending off-map toward Mars. The map's border is decorated with angular Pre-Flood Enochian script — geometric, alien, beautiful. The overall aesthetic blends Renaissance cartography with alien geometric precision. The colour palette is aged cream parchment, black ink line-work, golden sigil glow, and faint teal connection lines. Cartographic art. No modern text — only symbolic notation. No real-world modern labels.

---

### Style Notes
- **Mood:** Ancient intelligence mapped with modern precision, the global scope of the Watcher legacy
- **Palette:** Cream parchment, black ink, golden sigils, teal connection lines
- **Composition:** Full global map, twenty sigils distributed, network lines connecting
- **Aspect Ratio:** 16:9 landscape
